## ID：P-2026-01-2.1-001
**知识点**：#匀变速直线运动基本概念,#位移,#速度,#加速度
**难度**：0.6
**题型**：单选题
**题目描述**：物体沿一直线运动，在 $$t$$ 时间内通过的路程为 $$s$$，它在中间位置 $$\frac{s}{2}$$ 处的速度为 $$v_1$$，在中间时刻 $$\frac{t}{2}$$ 时的速度为 $$v_2$$，则 $$v_1$$ 和 $$v_2$$ 的关系是
**选项**：
A. 当物体做匀加速直线运动时，$$v_1 > v_2$$
B. 当物体做匀减速直线运动时，$$v_1 > v_2$$
C. 当物体做匀加速直线运动时，$$v_1 < v_2$$
D. 当物体做匀减速直线运动时，$$v_1 < v_2$$
**答案**：AB
**解析**：设物体初速度为 $$v_0$$，末速度为 $$v$$。中间时刻的速度 $$v_2 = \frac{v_0 + v}{2}$$。对于匀变速直线运动，中间位置的速度 $$v_1 = \sqrt{\frac{v_0^2 + v^2}{2}}$$。由于 $$\frac{v_0^2 + v^2}{2} > \left(\frac{v_0 + v}{2}\right)^2$$，所以 $$v_1 > v_2$$，无论是匀加速还是匀减速都成立，故AB正确。

## ID：P-2026-01-5.2-003

**题型**：单选题
**题目描述**：如图所示，在斜面顶端的 $$A$$ 点以速度 $$v$$ 平抛一小球，经 $$t_1$$ 时间落到斜面上 $$B$$ 点处，若在 $$A$$ 点将此小球以速度 $$2v$$ 水平抛出，经 $$t_2$$ 时间落到斜面上的 $$C$$ 点处，以下判断正确的是（不计空气阻力）
**选项**：
A. $$t_2 = t_1$$
B. $$t_2 = 2t_1$$
C. $$C$$ 点是 $$B$$ 点到斜面底端距离的中点
D. $$C$$ 点是 $$B$$ 点到斜面底端距离的四等分点
**答案**：B
**解析**：设斜面倾角为 $$\theta$$，平抛运动的水平位移 $$x = vt$$，竖直位移 $$y = \frac{1}{2}gt^2$$，落到斜面上时满足 $$\tan\theta = \frac{y}{x} = \frac{gt}{2v}$$。对于第一次平抛，$$\tan\theta = \frac{gt_1}{2v}$$；对于第二次平抛，$$\tan\theta = \frac{gt_2}{2 \times 2v} = \frac{gt_2}{4v}$$。两式相等，得 $$\frac{gt_1}{2v} = \frac{gt_2}{4v}$$，解得 $$t_2 = 2t_1$$，故B正确。沿斜面方向的位移 $$s = \frac{x}{\cos\theta}$$，第一次 $$s_1 = \frac{v t_1}{\cos\theta}$$，第二次 $$s_2 = \frac{2v \times 2t_1}{\cos\theta} = 4s_1$$，所以 $$C$$ 点不是中点或四等分点，故CD错误。
