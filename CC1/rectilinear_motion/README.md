# 文件概述

该文件夹下放置高中人教版必修一中第一章节的高考真题，主要涉及
- 运动的描述、
- 参考系、
- 质点、
- 匀变速直线运动基本概念，
- 匀变速直线运动公式的使用、
- 匀变速直线运动推论的应用、
- 自由落体与竖直上抛、
- 直线运动中常见的图像
- 追及与相遇问题

## 文件描述
该目录下的文件信息为
- motion_desc_uniform_accel.md（运动的描述，匀变速直线运动基本概念）
- uniform_accel_linear_inference.md（匀变速直线运动的推论、初速度为0以及初速度不为0的推论）
- vertical_projectile_motion.md（自由落体与竖直上抛）
- linear_motion_graphs.md（匀变速直线运动中涉及图像问题）
- pursuit_and_encounter.md（匀变速直线运动中追及与相遇问题）
