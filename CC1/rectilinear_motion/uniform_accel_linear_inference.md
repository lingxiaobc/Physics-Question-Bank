## ID：P-2026-01-1.2-001
**知识点**：#自由落体,#时间间隔,#位移
**难度**：0.6
**题型**：填空题
**题目描述**：(1990·上海·18)一矿井深为125m，在井口每隔一定时间自由下落一个小球。当第11个小球刚从井口下落时，第1个小球恰好到达井底，则相邻两个小球开始下落的时间间隔为______s，这时第3个小球和第5个小球相距______m。（$$g$$取$$10\text{m/s}^2$$）
**答案**：0.50；35
**解析**：由运动学公式得 $$h = \frac{1}{2}gt^2$$，解得 $$t = 5s$$。相邻两个小球开始下落的时间间隔 $$\Delta t = \frac{t}{10} = 0.5s$$。当第11个小球开始下落时，第3个小球已下落8个时间间隔即4s，第5个小球已下落6个时间间隔即3s，两球相距 $$\Delta h = \frac{1}{2}g(4^2 - 3^2)\text{m} = 35\text{m}$$。

## ID：P-2026-01-1.2-002
**知识点**：#竖直上抛,#自由落体,#时间间隔
**难度**：0.6
**题型**：单选题
**题目描述**：(2019·新课标·18)如图所示，篮球架下的运动员原地垂直起跳扣篮，离地后重心上升的最大高度为$$H$$，上升第一个$$\frac{H}{4}$$所用时间为$$t_1$$，第四个$$\frac{H}{4}$$所用时间为$$t_2$$，不计空气阻力，则$$\frac{t_1}{t_2}$$满足
![运动员原地起跳扣篮的示意图](https://physics-image-1391307169.cos.ap-nanjing.myqcloud.com/P-2026-01-1.2-002-question.png)
**选项**：
A. $$1 < \frac{t_1}{t_2} < 2$$
B. $$2 < \frac{t_1}{t_2} < 3$$
C. $$3 < \frac{t_1}{t_2} < 4$$
D. $$4 < \frac{t_1}{t_2} < 5$$
**答案**：C
**解析**：运动员的竖直上抛运动的逆向等效为自由落体运动，即 $$H = \frac{1}{2}gt^2$$，全程所用时间 $$t = \sqrt{\frac{2H}{g}}$$。下落第一个$$\frac{H}{4}$$所用时间为$$t_2$$，满足 $$\frac{H}{4} = \frac{1}{2}gt_2^2$$，解得 $$t_2 = \sqrt{\frac{H}{2g}} = \frac{t}{\sqrt{2}}$$。下落前三个$$\frac{H}{4}$$所用时间满足 $$\frac{3H}{4} = \frac{1}{2}gt'^2$$，解得 $$t' = \sqrt{\frac{3H}{2g}} = \sqrt{\frac{3}{2}}t_2$$。则 $$t_1 = t - t' = \sqrt{2}t_2 - \sqrt{\frac{3}{2}}t_2 = (\sqrt{2} - \sqrt{\frac{3}{2}})t_2$$... 由此可知 $$\frac{t_1}{t_2} = \frac{\sqrt{2}-\frac{\sqrt{3}}{\sqrt{2}}}{\frac{1}{\sqrt{2}}} = 2 - \sqrt{3} + (2-1) = \sqrt{2}(\sqrt{2}-\sqrt{\frac{3}{2}})$$。实际计算：上升第一个$$\frac{H}{4}$$对应下落第四个$$\frac{H}{4}$$，时间$$t_1 = t - \sqrt{\frac{3H}{2g}}$$；上升第四个$$\frac{H}{4}$$对应下落第一个$$\frac{H}{4}$$，时间$$t_2 = \sqrt{\frac{H}{2g}}$$。故$$\frac{t_1}{t_2} = \frac{\sqrt{\frac{2H}{g}} - \sqrt{\frac{3H}{2g}}}{\sqrt{\frac{H}{2g}}} = 2 - \sqrt{3} \approx 3.27$$，故C正确。

## ID：P-2026-01-1.2-003
**知识点**：#匀变速直线运动公式的理解与运用,#位移,#时间间隔
**难度**：0.7
**题型**：单选题
**题目描述**：(2024·山东·3)如图所示,固定的光滑斜面上有一木板,其下端与斜面上A点距离为$$L$$,通过A点的时间间隔为$$\Delta t_1$$;若木板长度为$$L$$时上端到达A点的时间为$$t_1$$;若木板长度为$$2L$$时上端到达A点的时间为$$t_2$$,木板长度为$$2L$$时上端到达A点的时间为
![固定光滑斜面上木板运动示意图](https://physics-image-1391307169.cos.ap-nanjing.myqcloud.com/P-2026-01-1.2-003-question.png)
**选项**：
A. $$(\sqrt{3}-1):(\sqrt{2}-1)$$
B. $$(\sqrt{3}-\sqrt{2}):(\sqrt{2}-1)$$
C. $$(\sqrt{3}+1):(\sqrt{2}+1)$$
D. $$(\sqrt{3}+\sqrt{2}):(\sqrt{2}+1)$$
**答案**：A
**解析**：设木板下端到达A点的时间为$$t_A$$，由运动学公式$$L = \frac{1}{2}at_A^2$$。当木板长度为$$L$$时，上端到达A点时木板下端距斜面底端为$$2L$$，有$$2L = \frac{1}{2}at_1^2$$，解得$$t_1 = \sqrt{2}t_A$$，则$$\Delta t_1 = t_1 - t_A = (\sqrt{2}-1)t_A$$。当木板长度为$$2L$$时，上端到达A点时木板下端距斜面底端为$$3L$$，有$$3L = \frac{1}{2}at_2^2$$，解得$$t_2 = \sqrt{3}t_A$$，则$$\Delta t_2 = t_2 - t_A = (\sqrt{3}-1)t_A$$。故$$\Delta t_1 : \Delta t_2 = (\sqrt{2}-1):(\sqrt{3}-1)$$，即$$\Delta t_2 : \Delta t_1 = (\sqrt{3}-1):(\sqrt{2}-1)$$，故A正确。

**解析**：(1)$$v-t$$图像如图所示。
![v-t图像解析图](https://physics-image-1391307169.cos.ap-nanjing.myqcloud.com/P-2026-01-1.2-004-analysis.png)
(2)设刹车前汽车匀速行驶时的速度大小为$$v_0$$，则$$t_1$$、$$t_2$$时刻的速度分别为$$v_1$$、$$v_2$$，在$$t_2$$时刻后汽车在4s内的位移为24m，设其加速度大小为$$a$$，取$$\Delta t = 1s$$，设汽车在$$t_2 + 3\Delta t \sim t_2 + 4\Delta t$$时间内未停止，$$v_2 + (n-1)\Delta t \cdot a - \frac{1}{2}a\Delta t^2$$，$$s_1 - s_4 = 3a(\Delta t)^2$$，$$s_4 = v_2\Delta t - \frac{1}{2}a(\Delta t)^2 - 4a\Delta t$$，联立并代入已知数据解得$$v_2 = -\frac{17}{6}\text{m/s}$$，这说明在$$t_2 + 4\Delta t$$刻前，汽车已经停止,因此不成立。若汽车在$$t_2 + 3\Delta t \sim t_2 + 4\Delta t$$时间内停止，由运动学公式$$s_1 = v_2\Delta t - \frac{1}{2}a(\Delta t)^2$$，$$s_4 = v_2 - 4a\Delta t$$，联立解得$$v_2 = 8\text{m/s}$$，$$a = 2\text{m/s}^2$$。
