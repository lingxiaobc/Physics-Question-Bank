# Physics Question Bank

## 项目说明
本项目为高中物理题库，从 https://github.com/lingxiaobc/Physics-Question-Bank.git 拉取。

## 题库覆盖范围

**整个高中物理，且仅限高中物理。** 涵盖人教版高中物理全部内容：

- **必修一**：直线运动、相互作用与牛顿运动定律
- **必修二**：曲线运动与万有引力、机械能守恒定律
- **必修三**：静电场、恒定电流、电磁感应初步
- **选择性必修一**：动量守恒定律、机械振动与机械波、光
- **选择性必修二**：磁场、电磁感应、交变电流
- **选择性必修三**：热学、原子物理与核物理

## 题目格式规范（强制）

**每道题目必须严格按照以下 Markdown 格式编写，不得偏离。**

### 分隔规则

每道题目之间**必须**用 `===分割线===` 分隔，独占一行。

### 完整模板

```
===分割线===
## ID：{题目编号}
**知识点**：{标签列表}
**难度**：{难度值}
**题型**：{题型名称}
**题目描述**：{题目正文}
**选项**：
 A. {选项A}
 B. {选项B}
 C. {选项C}
 D. {选项D}
**答案**：{正确答案}
**解析**：{详细解析}
```

### 各字段规范

#### 1. ID（题目编号）
- 格式：`## ID：P-YYYY-XX-Y.Z-NNN`
- 组成部分：
  - `P` — 物理题库前缀
  - `YYYY` — 命制年份，如 `2026`
  - `XX` — 教材册别：`01`=必修一，`02`=必修二，`03`=必修三，`S1`=选择性必修一，`S2`=选择性必修二，`S3`=选择性必修三
  - `Y.Z` — 章节编号，如 `1.1`=第一章第一节
  - `NNN` — 三位数字编号，从 `001` 开始递增
- 示例：`## ID：P-2026-01-1.1-001`

#### 2. 知识点（标签）
- 格式：`**知识点**：#标签1,#标签2,#标签3`
- 每个标签以 `#` 开头，多个标签用英文逗号 `,` 分隔
- 标签名称使用中文，简洁描述核心考点

#### 3. 难度
- 格式：`**难度**：0.X`
- 取值范围：`0.1` ~ `0.9`
- `0.9`=极简单，`0.8`=简单，`0.7`=较简单，`0.6`=中等偏易，`0.5`=中等，`0.4`=较难，`0.3`=难题

#### 4. 题型
- 格式：`**题型**：{题型名称}`
- 题型名称必须在：`单选题` | `多选题` | `填空题` | `解答题` | `计算题`

#### 5. 题目描述
- 格式：`**题目描述**：{正文}`
- 若来源于高考真题，在正文开头标注来源：`(YYYY·省份·题号)`
- 正文后可紧跟图片

#### 6. 选项
- 单选题、多选题必填，填空题、解答题省略此字段
- 格式：
  ```
  **选项**：
   A. {选项内容}
   B. {选项内容}
   C. {选项内容}
   D. {选项内容}
  ```
- 每个选项独占一行，大写字母 `A.` `B.` `C.` `D.` 加一个空格
- 选项内可包含 `$$...$$` 数学公式或图片链接

#### 7. 答案
- 格式：`**答案**：{答案内容}`
- 单选题：单个字母如 `C`
- 多选题：多个字母如 `AD`
- 填空题：数值或表达式，多个答案用分号分隔如 `17；17.9`
- 解答题：最终结果如 `(1)20m/s；(2)680m`

#### 8. 解析
- 格式：`**解析**：{解析正文}`
- 包含完整推导过程、公式、计算结果
- 可包含解析配图

### 数学公式规范

- 使用 `$$...$$` 包裹公式
- 示例：`$$\frac{v_0}{g}$$`、`$$\sqrt{2}$$`、`$$v_0$$`、`$$\text{m/s}^2$$`、`$$\bar{v}$$`、`$$\Delta x$$`

### 图片存储规范

- 项目所有图片统一存放在 `images/` 目录下
- 图片目录结构**必须**与题目文件的目录结构一一对应
- **同一份 `.md` 文件中所引用的图片，必须存放在 `images/` 下对应的同一个子目录中**

示例对应关系：

| 题目文件路径 | 图片存放路径 |
|---|---|
| `CC1/rectilinear_motion/motion_desc_uniform_accel.md` | `images/CC1/rectilinear_motion/motion_desc_uniform_accel/` |
| `CC1/rectilinear_motion/uniform_accel_linear_inference.md` | `images/CC1/rectilinear_motion/uniform_accel_linear_inference/` |
| `CC1/rectilinear_motion/vertical_projectile_motion.md` | `images/CC1/rectilinear_motion/vertical_projectile_motion/` |

- 图片引用格式：`![描述文字](../images/{相对路径}/{文件名}.png)`
- Markdown 文件在 `CC1/` 下时，相对路径以 `../images/` 开头
