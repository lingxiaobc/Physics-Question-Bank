# Physics Question Bank

## 项目说明
本项目为物理题库（Physics Question Bank），从 https://github.com/lingxiaobc/Physics-Question-Bank.git 拉取。

## 搜索规则（强制）

**主代理严禁直接搜索项目内容。** 所有文件查找、代码搜索、内容定位等操作，必须委托 `project-searcher` 子代理完成。

- 需要查找文件？→ 调用 `project-searcher`
- 需要搜索代码？→ 调用 `project-searcher`
- 需要定位函数/类/变量？→ 调用 `project-searcher`
- 需要了解项目结构？→ 调用 `project-searcher`

主代理仅可通过 Read 工具读取已知路径的文件（如 CLAUDE.md、settings.json 等配置文件），但不得用于探索性搜索。
